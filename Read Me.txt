VCA Ops Portal
Author: Harold Kammermeyer

To launch double click on 'VCA Ops Portal.cmd'
- If you're unable to launch via .cmd, right click on 'VCAHospLauncher.ps1' and 'Run with PowerShell'

Requirements:
 - Windows 8+/Windows Server 2012+
 - Powershell v4 or v5
Powershell modules:
 Remote Server Administration Tools (RSAT) - https://www.microsoft.com/en-us/download/details.aspx?id=45520
 - ActiveDirectory
 - DnsServer

Applications:
 - VMWare Remote Console

Included:

  - VMware.VimAutomation.Sdk"
  - VMware.VimAutomation.Common"
  - VMware.Vim"
  - VMware.VimAutomation.Cis.Core"
  - VMware.VimAutomation.Core"
 - HPEiLOCmdlets - https://www.powershellgallery.com/packages/HPEiLOCmdlets/
Binaries
 - Putty
